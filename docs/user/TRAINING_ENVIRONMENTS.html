📍 Training Environments

Tell us where you train so we know what equipment you have to work with

Overview

Your fitness profile (age, experience, goals) doesn't change much day to day — but where you train might. Maybe you lift at a commercial gym on weekdays, do bodyweight circuits at home on weekends, and drop into a CrossFit box occasionally. Training Environments let you save each of these setups separately, with its own name, type, and equipment list, so a program built for "Home" doesn't assume you have a squat rack.

✓ Add as many as you need — a home setup, a commercial gym, a bodyweight-only routine — and pick the right one whenever you're ready to generate a program.

Before You Start

  • You must be logged into your MyGym account
  • You don't need to have completed onboarding's Training Environments step first — you can add environments anytime
  • Think about the different places you actually train and what's available at each

How to Add a Training Environment

  1. Open the user menu in the header and click Training Environments
  2. Click Add Environment
  3. Give it a Name (e.g. "Home Gym", "Downtown CrossFit")
  4. Choose a Type: Commercial Gym, Home, Bodyweight Only, CrossFit Box, or Other
  5. Check off the equipment available there (barbell, squat rack, dumbbells, kettlebell, resistance bands, pull-up bar, bench, cardio machine, cable machine, or none)
  6. Optionally check Set as default environment
  7. Click Save Environment
💡 Adding one during onboarding? The onboarding wizard has the same form embedded in its "Training Environments" step. It has its own Save Environment button — separate from the wizard's Back/Next controls. Click Save Environment to add it to your list before moving to the next step.

Managing Your Environments

From the Training Environments page, each saved environment shows as a card with:

  • Edit — Update the name, type, equipment, or default status
  • Delete — Remove it permanently (you'll be asked to confirm)
  • Generate Program — Start creating a workout program based on this environment (see below)

Default Environment

Marking one environment as default is just a convenience flag — only one environment can be the default at a time. Setting a new one automatically un-defaults the previous one.

Generating a Program Coming Soon

Clicking Generate Program on any environment opens a short questionnaire so we can tailor that specific program:

  1. Days per Week and which specific days you plan to train
  2. Session Duration — how long you have per workout
  3. Start Date — when the program should begin
  4. Focus Areas — push, pull, legs, core, cardio, flexibility, or full body
  5. Weight Unit — kilograms or pounds
  6. Available Weight Increments — the smallest jumps you can actually load (e.g. 1.25, 2.5, 5)
  7. Progression Style — consistent (same every week) or variable (changes week to week)

These preferences are specific to this program-generation attempt — they aren't saved to your profile or environment, since they might be different the next time you generate a program. Submitting the form today shows a "coming soon" message; program generation itself hasn't been built yet, but your preferences round-trip through the same request the real feature will use.

Troubleshooting

❓ I filled in the form but my environment isn't saved

Check that you clicked Save Environment — filling in fields alone doesn't save anything until that button is clicked. If you were in the onboarding wizard, make sure you clicked Save Environment before Next; the wizard's Next button is disabled while the form is open specifically to prevent this.

❓ Can I have more than one environment marked as default?

No — only one at a time. Checking "Set as default" on a new or edited environment automatically clears the flag from whichever one had it before.

❓ Why can't I generate a real program yet?

Program generation is still being built. The questionnaire is fully functional and your answers are validated, but submitting it currently returns a "not yet implemented" message instead of an actual program.

❓ I deleted an environment by accident

Deletion is permanent and can't be undone — you'll need to re-add it with the same details. We ask for confirmation before deleting specifically to help avoid this.

Tips & Tricks

  • 💡 Keep environment names specific ("Home — Garage Gym" vs. just "Home") if you might add more than one home setup later.
  • 💡 Only check the equipment you actually have reliable access to — an accurate list means better exercise recommendations once program generation ships.
  • 💡 It's fine to finish onboarding with zero environments — add your first one whenever you're ready from the Training Environments page.
